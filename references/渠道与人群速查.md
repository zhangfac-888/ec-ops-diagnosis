# 渠道与人群速查（电商运营诊断 · 横切查表）

> 主图诊断 / 详情页诊断 / 人群验证 三段都会用到的**横切查表**：这个平台是什么气质、这群人靠什么驱动购买、哪些词平台不让写。
> 用法：判"渠道适配""该主打什么卖点""文案合不合规"时，查这三张表，结论挂 `basis`。

---

## 一、平台气质矩阵（判"渠道适配"）

| 平台 | 视觉方向 | 文案密度 |
|---|---|---|
| 天猫 | 质感、品牌感、真实场景、留白 | 低到中 |
| 淘宝 | 场景清楚、卖点明确、兼顾质感 | 中 |
| 拼多多 | 强识别、强卖点、直接转化 | 中到高 |
| 京东 | 参数、品质、效率、信任 | 中 |
| 抖音 | 使用瞬间、场景感、内容感 | 中 |
| 小红书 | 生活方式、真实体验、审美 | 低到中 |
| 亚马逊 | 白底、功能、参数、对比 | 中 |
| 独立站 | 品牌调性、留白、杂志感 | 低 |

> 同一张图，天猫要留白克制、拼多多要强卖点直给——"好不好"先看落在哪个平台。

## 二、购买决策驱动六类（判"该主打什么"）

| 驱动类型 | 视觉重点 | 文案重点 |
|---|---|---|
| 颜值驱动 | 场景、色彩、材质、生活方式 | 风格、人群、拥有感 |
| 功能驱动 | 使用动作、结构证明、对比 | 解决什么问题 |
| 参数驱动 | 信息图、数字、规格 | 参数转用户利益 |
| 信任驱动 | 细节、材质、检测/服务 | 依据明确 |
| 价格驱动 | 产品放大、卖点直给 | 简短强利益 |
| 高客单价 | 留白、克制、真实摄影 | 少字、品质感 |

> 与人群验证配合：同一卖点对不同人群落点不同（如"第三方检测"对信任驱动的精致妈妈是硬通货，对价格驱动的人几乎无感）。先判这群人靠什么驱动，再判卖点打不打得动。

## 三、平台合规禁用词（判"文案能不能写"）

下列词**多数平台违规或夸大、不可写**（与"真实性红线"并列的"合规红线"）：

> 爆款 · 全网 · 第一 · 最 · 永久 · 100% · 食品级 · 0甲醛 · 环保认证 · 防霉 · 专用 · 承重XX斤 · 医用 · 治疗/疗效 · 国家级 · 顶级

- 参数/容量/时长估算用"约"。
- 对比图用"普通款 vs 本款"，不写"劣质品/垃圾款"、不丑化、不点竞品牌。
- 卖点转化链：`产品事实 → 使用场景 → 用户痛点/期待 → 买家利益 → 合规短文案`。

---

> ⚠️ 注意区分两类红线：**真实性红线**（本诊断核心：不编频率、不预测业绩、缺数据降级）管"会不会自欺"；**合规红线**（本表）管"会不会被平台罚"。两类都要守。
