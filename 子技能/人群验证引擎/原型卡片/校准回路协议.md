# 校准回路协议

> 版本：v1.0
> 适用范围：八大人群原型卡片的所有生成变体
> 使用时机：AI生成变体卡片后、人工修改底座后、投入实际任务前

---

## 一、四关检查（按顺序执行）

### 关0 — 归属前置：表面信号 ≠ 底层逻辑

在确定人群归属之前，先用消费逻辑而不是人口学标签来判定。

**核心判据（三条，逐一过）**

| 判据 | 偏白领/城市方向 | 偏蓝领/小镇方向 |
|-----|----------------|----------------|
| 消费叙事 | 自我投资/效率/形象/犒劳 | 能用/不浪费/给家里/省着 |
| 城市关系 | 城市已重塑生活方式 | 城市只是工作地点 |
| 信任来源 | 测评/成分/博主/内容平台 | 熟人/邻居/子女/老品牌 |

**常见误判（快速排查）**

| 表面信号 | 不等于 | 判断点 |
|---------|--------|-------|
| 写字楼上班 | ≠ 新锐白领 | 外包客服/前台/保安/驻场运维/物业/仓储/配送 = 都市蓝领；关键：有无职业晋升叙事 |
| 住一线城市 | ≠ 白领消费逻辑 | 人在一线但仍是拼多多/省钱/熟人推荐/家里优先 = 都市蓝领/小镇青年 |
| 收入不低 | ≠ 资深中产 | 小老板/包工头/县城生意人；关键："体面/时间效率" vs "划算/不被骗/熟人说好" |
| 年龄年轻 | ≠ Z世代 | 22岁工厂工人/外卖骑手/县城服务员；关键：有无圈层消费/审美表达/社交货币 |
| 退休老人 | ≠ 都市银发 | 三四线/乡镇退休 = 小镇中老年；关键：微信+京东+社区 vs 集市+邻居+线下熟人 |
| 有孩子 | ≠ 精致妈妈 | 蓝领/小镇也有孩子；关键："教育焦虑/妈妈群比较" vs "家里该花/别亏待孩子" |
| 偶尔买贵东西 | ≠ 高L档 | 低L档也会为孩子/健康/面子咬牙一次；关键：高价是常态 vs 一次性特殊消费 |

**蓝领驻写字楼的归属规则**

保安、保洁、食堂、仓储、运维、客服外包、物业、司机、配送、驻场技术工——即使在大公司，仍归都市蓝领。

判断句：**时间被排班控制 + 身体/情绪被工作消耗 + 消费以"能用/给家里"为主** → 归都市蓝领，不管工作地点。

**漂移态（特殊标注）**

在大公司待3-5年的低职级行政/助理：L档不变，但消费语言可能已部分向白领靠拢（开始用"平替"、偶尔刷小红书）。此类混合态不强制归档，卡片里注明"元心态和语气需人工校准"。

**如仍不确定**：生成两张候选人群卡片，盲测哪张更接近，取更接近的。

### 关1 — 生成前：参数合法性

在生成卡片之前，先确认参数组合合法。

```
□ 查合法性矩阵：档位×L档组合是否标注 ✅ ？（⚠️少见可生成但需注明）
□ P轴类型是否为该人群5种P之一？
□ 人群归属是否清晰？（蓝领/小镇青年 用边界规则判定现居地；银发/中老年 用城市档判定）
□ 如有性别子原型需求，是否已加载对应附录差异？
```

**如参数非法**：停止生成，调整参数至合法组合。不要强行生成非法组合。

### 关2 — 生成后：自检（逐项过）

生成完毕，对卡片逐项检查。

#### A. 禁用词扫描

对照该人群禁用词表，全文搜索。**出现一个即失真，整段重写。**

| 人群 | 高频失真词（快速扫） |
|------|-----------------|
| Z世代 | 实用 / 高性价比 / 品质生活 / 成熟 / 理性消费 |
| 新锐白领 | 轻奢 / 品质生活 / 仪式感拉满 / 种草成功 / 精致 |
| 精致妈妈 | 轻奢 / 种草成功 / 颜值在线 / 仪式感拉满 / 精致 |
| 资深中产 | 轻奢 / 仪式感 / 种草成功 / 颜值在线 / 爆款 / 闭眼入 |
| 都市蓝领 | 轻奢 / 仪式感 / 种草 / 颜值 / 品质生活 / 精致 / 成分党 |
| 都市银发 | 种草 / 颜值 / 仪式感 / 网红款 / 精致 / 入手 / 闭眼入 |
| 小镇青年 | 轻奢 / 仪式感 / 品质生活 / 颜值党 / 精致 / 消费升级 |
| 小镇中老年 | 轻奢 / 仪式感 / 种草 / 颜值 / 爆款 / 网红款 / 消费升级 |

#### B. 三轴职责边界

```
□ 档位轴：是否混入了消费金额、购买力描述、决策风格？（→ 移出到L/P轴）
□ L轴：是否混入了决策风格、信任来源描述？（→ 移出到P轴）
□ P轴：是否混入了消费金额、月均支出？（→ 移出到L轴）
```

#### C. L轴口径核查

```
□ 月均消费 = 电商平台个人下单支出，不含房租/餐饮/线下
□ 他人代买不计入本人L档（若为P5子女依赖型/P5囤货涉及代买，需注明）
□ 消费描述是否与L档区间匹配？（L1 <400 / L2 400-1k / L3 1k-3k）
```

#### D. 语域检查

```
□ 对照该人群语气锚点：卡片语气是否和锚点例句属于同一语感？
□ 是否出现该人群不该有的句式或词汇？
   - 蓝领说话像白领？银发说话像年轻人？小镇中老年夹网络词？
□ 内心独白是否过于文学化？（应是日常口语，不是散文）
```

#### E. 核心矛盾张力

```
□ 核心矛盾是否有真实张力？（两个相互冲突的需求或身份，不是一个单一困境）
□ 消费合理化是否符合该人群的逻辑？
   - 蓝领：家里/孩子=正当；自己要回答"真的需要吗"
   - 银发：健康/子孙辈=正当；自己的消费要"对身体好"
   - 小镇青年：家里/孩子=正当；自己要"就这点钱，开心就行"
```

### 关3 — 盲测：近邻可区分性

> 只对重要卡片或边界模糊情况执行。快速版：自问能不能从描述推出档×L×P。

#### 快速盲测步骤

1. 遮盖卡片标题和标签
2. 只看生活切片和语气
3. 猜：这张卡片是哪个人群？档几？L几？P几？
4. 猜对说明区分度足够；猜不对说明某个轴特征不够突出

#### 近邻对照（重点检查的组合）

| 容易混淆的对 | 区分维度 |
|------------|---------|
| 都市蓝领 vs 小镇青年 | 现居地（大城市务工 vs 老家生活）；语气（极简电报 vs 随和松弛） |
| 都市银发 vs 小镇中老年 | 城市层级（一二线退休 vs 三四线县城）；数字化程度 |
| 新锐白领 vs Z世代 | 年龄/职业阶段；消费合理化（投资自己 vs 纯粹圈子认同） |
| 精致妈妈 vs 资深中产 | 性别/消费驱动（孩子焦虑 vs 值不值得判断） |
| 同人群 L2 vs L3 | 客单价行为、品牌认知程度、大促参与方式 |
| 同人群 P2 vs P4 | 信任来源（多渠道验证 vs 认定品牌不换） |
| 资深中产P1 vs Z世代P1（同P档跨人群） | 触发机制（需求自发冒出 vs 内容/算法推送）；事后语气（买完就是了 vs 截图发帖分享） |

## 二、修复路由（按失败类型）

| 失败类型 | 定位方法 | 修复动作 |
|---------|---------|---------|
| 禁用词出现 | 全文搜索禁用词表 | 整段重写，对照语气锚点调语感 |
| 档位轴混入L/P内容 | 找消费金额、购买力描述、决策路径描述 | 提取相关内容移至L/P轴对应字段 |
| L轴混入P内容 | 找"信任谁/怎么决策" | 移至P轴，L轴只留消费行为和金额 |
| 语域失真 | 对照语气锚点朗读比较 | 替换失真段落，逐句校准语气 |
| L档口径错误 | 核查月均计算口径 | 重算L档，注意他人代买规则 |
| 核心矛盾无张力 | 读核心矛盾：是否只是一个困难而不是两股力量 | 补充相反方向的驱动力 |
| 盲测猜不中人群 | 找缺失的特有标记 | 加一条该人群专属的生活切片或语气特征 |
| 盲测猜不中L档 | 找客单价和购买决策行为 | 强化该L档的具体消费行为描述 |
| 盲测猜不中P档 | 找决策路径和信任来源 | 强化该P档的触发动作 |

## 三、合格标准

一张变体卡片通过三关后，满足以下条件视为合格：

```
✅ 关0：归属判断——消费逻辑三判据过，无误判
✅ 关1：参数合法（矩阵✅或⚠️）
✅ 关2A：禁用词零命中
✅ 关2B：三轴内容无串位
✅ 关2C：L轴口径正确
✅ 关2D：语域与锚点匹配
✅ 关2E：核心矛盾有两股力量
✅ 关3：可被盲测识别（或判断近邻对中可区分）
```

**特殊情况备注**：
- ⚠️少见组合：合法但生成时加注"[少见组合，请人工确认合理性]"
- P5子女依赖型（小镇中老年）/ P5囤货型（都市银发）含代买：卡片内注明L档口径
- 性别子原型：在关2D额外对照附录语气锚点，不只看主原型

## 四、快速检查卡（一页打印版）

```
归属判断（关0）
□ 消费叙事：投资/效率 vs 能用/给家里
□ 城市关系：生活方式已重塑 vs 只是工作地点
□ 信任来源：内容平台/测评 vs 熟人/邻居/老品牌
□ 无常见误判（写字楼≠白领 / 退休≠都市银发 / 有孩子≠精致妈妈……）

生成前（关1）
□ 合法性矩阵：档×L = ✅/⚠️
□ P轴合法
□ 人群边界明确

生成后（自检5项）
□ 禁用词：0命中
□ 三轴不串位
□ L口径正确（他人代买≠本人L档）
□ 语域：对照锚点朗读通过
□ 核心矛盾：两股力量存在

盲测（重要卡片执行）
□ 遮标签，能猜中人群
□ 能猜中L档
□ 能猜中P档
□ 与近邻可区分
```
