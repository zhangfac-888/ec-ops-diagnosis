# P轴全局映射表

> 版本：v1.0
> 用途：确保跨原型 P轴使用同一套槽位 ID（P1-P5），同一张系统中不同原型可以比较。
> 结构：全局构念（不变）→ 各原型本地标签（允许差异）

---

## 设计原则

P轴有两层：

1. **全局构念层**（P1-P5 槽位 ID + 底层心理机制）  
   跨所有原型成立，不依赖具体人群。

2. **原型本地层**（本地标签 + 该原型的具体行为描述）  
   同一构念在不同人群里的表达方式不同，需要用该人群自己的语言描述。

> **严禁**：用全局构念标签直接写入卡片。卡片里必须用原型本地标签。

---

## 全局槽位定义

| 槽位 | 底层构念 | 核心机制 |
|-----|---------|---------|
| **P1** | 冲动触发型 | 情境刺激→即时下单，不经过理性核查阶段 |
| **P2** | 信息核查型 | 需要主动收集信息、反复验证后才能下单 |
| **P3** | 社交参照型 | 决策前置条件是"有同圈子的人用了说好" |
| **P4** | 品牌锚定型 | 认定了某个品牌/品类，不重新做研究，直接复购 |
| **P5** | 非即时独立型 | 不在当下独立做购买决策（延迟集中/依赖他人） |

---

## 各原型本地标签映射

### P1：冲动触发型

| 原型 | 本地标签 | 触发场景特征 |
|-----|---------|------------|
| 精致妈妈 | 冲动型 | 深夜刷到直接拍；明天截图发闺蜜"我昨晚又乱买了" |
| 新锐白领 | 冲动型 | 刷到种草直接加购，晚上再刷一遍就下单 |
| 资深中产 | 冲动型 | 想到需要了直接搜，决策快，靠第一直觉和品牌印象 |
| 都市银发 | 冲动型 | 刷抖音直播间看着好就买了，或超市看到打折冲动买 |
| Z世代 | 冲动型 | 刷到就买，付款动作极快；买完发小红书"入手了" |
| 小镇青年 | 冲动型 | 刷直播间看着好直接拍，买完说"就这点钱" |
| 都市蓝领 | 冲动型 | 直播间看着便宜/实用，直接拍；买完觉得也没亏多少 |
| 小镇中老年 | 被动冲动型 | 子女/老伴说要买，当场就答应了；或看到低价促销当下直接买 |

> **注意**：小镇中老年 P1 是"被动冲动"——触发力量来自外部（子女/老伴/低价），不是自己刷到的内容。

### P2：信息核查型

| 原型 | 本地标签 | 核查渠道/方式 |
|-----|---------|-------------|
| 精致妈妈 | 谨慎型 | 加购物车放一周；发闺蜜群问"有人用过吗"；反复看中差评 |
| 新锐白领 | 谨慎型 | 看成分表；找三个不同来源测评都说好；加购物车等说服自己 |
| 资深中产 | 谨慎型 | 认真看评测，对比几个选项，确认最优解才下单 |
| 都市银发 | 谨慎型 | 问了好几个老姐妹，打电话给子女问；反复确认才下单 |
| Z世代 | 研究型 | 刷大量测评视频，看评论区，确认"这个圈子里口碑好"再下手 |
| 小镇青年 | 谨慎型 | 看评价/销量，问一下身边朋友，确认了才买 |
| 都市蓝领 | 谨慎型 | 先搜一圈确认最低价，等价格到位再买；不急 |
| 小镇中老年 | 谨慎型 | 先问老伴/邻居，确认有人用过了才动手 |

> **注意**：Z世代 P2 叫"研究型"而非"谨慎型"——行为相同（信息核查），但动机是"圈子口碑"而非"理性回避风险"，语感完全不同。

### P3：社交参照型

| 原型 | 本地标签 | 参照圈子 |
|-----|---------|--------|
| 精致妈妈 | 从众型 | 闺蜜/同事/同学妈妈；主动问"你们都买哪家" |
| 新锐白领 | 从众型 | 同事/闺蜜；确认真实使用者说好，买同款 |
| 资深中产 | 从众型 | 同事/商业圈子；认为"圈子里都在用的不会太差" |
| 都市银发 | 从众型 | 邻居/老年活动圈；"大家都用了应该没问题" |
| Z世代 | 圈层型 | idol在用/KOL圈子在用/好朋友在用；同圈子的人用了才入手 |
| 小镇青年 | 从众型 | 朋友/同事/邻居推荐；"大家都买的应该没问题" |
| 都市蓝领 | 从众型 | 同事/工友推荐；"他用了觉得好，我跟着买" |
| 小镇中老年 | 从众型 | 邻居/同龄人；"大家都用的应该没问题" |

> **注意**：Z世代 P3 叫"圈层型"而非"从众型"——不只是"大家都用"，而是特定圈层认同感驱动（IP粉/小众圈子/审美圈），语感完全不同。

### P4：品牌锚定型

| 原型 | 本地标签 | 换品牌的门槛 |
|-----|---------|-----------|
| 精致妈妈 | 品牌型 | 认定几个品牌，其他基本不看；换品牌需踩雷/涨价/被强推荐 |
| 新锐白领 | 品牌型 | 认定品牌，出了新品直接买；换品牌要有明确理由 |
| 资深中产 | 品牌型 | 认定品牌直接回购，不重新研究；需踩雷或值得信的人强推 |
| 都市银发 | 品牌型 | 认定几个老品牌，其他不看；换品牌要有"自己踩过雷/子女极力推荐" |
| Z世代 | 品牌型 | 认定某品牌的理念/审美，出新品直接看；比其他原型更看重品牌"有没有在圈子里" |
| 小镇青年 | 品牌型 | 认定"质量还行"的品牌，不换；主要是怕买到假货/次货 |
| 都市蓝领 | 品牌型 | 认定"不会是假的"的品牌，回购；换品牌要踩过一次才肯换 |
| 小镇中老年 | 品牌型 | 认定老品牌；"以前的牌子用着放心"；不主动接受新品牌 |

> **注意**：各原型 P4 核心都是"认定后复购"，但触发因素不同：银发/中老年靠"用了多年的老牌"，Z世代靠"品牌在圈子里的身份感"，蓝领/小镇靠"不是假货的保障"。

---

### P5：非即时独立型

| 原型 | 本地标签 | 延迟/依赖机制 |
|-----|---------|-------------|
| 精致妈妈 | 大促型 | 平时看到好东西加收藏不买；双11/618前做清单；凑满减 |
| 新锐白领 | 大促型 | 购物车积累着，等618/双11统一处理；平时收藏，不急着买 |
| 资深中产 | 效率型 | 喜欢年货节/618批量采购；集中一个节点处理比零散买更高效 |
| 都市银发 | 囤货型 | 打折了就多买几个备着；子女大促帮买一大批；"放着不亏的" |
| Z世代 | 大促型 | 购物车存着等大促；但也会抢首发限量款（即时性高） |
| 小镇青年 | 大促型 | 双11/618存着等，或等直播间出活动价；不在大促买觉得亏 |
| 都市蓝领 | 计划型 | 攒着，有明确目的再买；不主动逛，想到需要了才搜 |
| 小镇中老年 | 子女依赖型 | 需要什么先打电话问子女；子女说买什么就买什么；子女帮着下单 |

> **注意**：P5 是本轴分化最大的槽位。
> - 大促型/效率型/囤货型：共同点是"不在当下买"，驱动是价格节点或批量效率。
> - 计划型：驱动是"需要确认"，不是节点。
> - 子女依赖型：驱动是"决策权外包"，与其他四个逻辑完全不同——不是时间延迟，是决策主体转移。
> 
> **生成规则**：选择 P5 时，卡片里必须使用原型本地标签，不写"大促型"在蓝领卡片里。

## 使用说明

1. **生成卡片时**：P轴参数从本文件查找"原型本地标签"，复制到卡片里
2. **跨原型比较时**：用全局槽位 ID（P1-P5）而不是本地标签
3. **发现新决策模式时**：先判断它属于哪个槽位，再更新对应原型的本地描述，不新建槽位
4. **P5 特别提醒**：子女依赖型（小镇中老年）和大促型/计划型不是同一机制，但同在 P5 槽位——这是有意的设计，P5 的共同点是"非当下独立决策"
