# 选归属 · 关0 判据

> 来源：`../原型卡片/校准回路协议.md` 关0（本 Skill 内置）。本文件是引擎"按真实客群选有份量人群变体"那一步的判据，**禁人口学倒推**（红线③）。
> 用法：对照真实客群构成里占得上份量的人群，逐一用三判据确认归属，不是把 8 张卡全跑一遍。

## 三判据（逐一过）

| 判据 | 偏白领/城市方向 | 偏蓝领/小镇方向 |
|-----|----------------|----------------|
| 消费叙事 | 自我投资/效率/形象/犒劳 | 能用/不浪费/给家里/省着 |
| 城市关系 | 城市已重塑生活方式 | 城市只是工作地点 |
| 信任来源 | 测评/成分/博主/内容平台 | 熟人/邻居/子女/老品牌 |

## 常见误判（禁表面信号倒推）

| 表面信号 | ≠ | 判断点 |
|---------|---|-------|
| 写字楼上班 | ≠ 新锐白领 | 外包客服/前台/保安/驻场运维 = 都市蓝领；看有无晋升叙事 |
| 住一线城市 | ≠ 白领逻辑 | 人在一线仍拼多多/省钱/熟人推荐 = 蓝领/小镇青年 |
| 收入不低 | ≠ 资深中产 | 小老板/包工头；"体面/效率" vs "划算/不被骗" |
| 年龄年轻 | ≠ Z世代 | 22岁工厂工/骑手；看有无圈层消费/审美表达 |
| 退休老人 | ≠ 都市银发 | 三四线退休 = 小镇中老年；微信+京东 vs 集市+邻居 |
| 有孩子 | ≠ 精致妈妈 | 看"教育焦虑/妈妈群比较" vs "家里该花/别亏待孩子" |
| 偶尔买贵 | ≠ 高L档 | 看高价是常态 vs 一次性特殊消费 |

## 选人群规则

1. 先看用户给的**真实客群构成**，列出占得上份量的人群。
2. 每个用三判据 + 误判表确认归属，必要时用城市档/边界规则定位（蓝领 vs 小镇青年用现居地；银发 vs 中老年用城市档）。
3. 占比极小、对结论无影响的人群可不跑——但要说明"略过谁、为什么"。
4. **如真实客群构成缺失** → 不许靠人口学猜人群占比，转入降级流程（见 `输出模板.md` 降级模板）。
