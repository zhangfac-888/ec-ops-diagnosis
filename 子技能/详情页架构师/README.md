# 详情页架构师 Skill v2.0.0

> 一个用方法论压住 AI 编造倾向的电商详情页工具。
>
> 核心约束：**没有数据不开工**。AI 不会假装替你做你应该做的市场调研。

---

## 这个 Skill 解决什么问题

电商运营写详情页有两种常见困境：

1. **AI 帮我写 → 一看就是 AI 味，全是套话**：因为 AI 没有你的真实数据，只能凭通用品类常识凑。
2. **我自己写 → 没有方法论结构，写出来散乱**：缺一套清晰的「先写什么后写什么」的骨架。

本 Skill 的设计是把这两个问题分开解决：

- **数据**：强制你提供 5 项必填数据，缺了不开工——让 AI 写的不是「通用废话」而是「基于你真实情报的方案」。
- **方法论**：内置卖点挖掘四维法 + 12-15 屏结构 + 因果文案公式——把骨架塞给 AI，让它按章节生成。

## 安装

### Claude Code（推荐）

1. 解压 zip 到 `~/.claude/skills/详情页架构师-skill/`
2. 重启 Claude Code，触发词「写详情页 / 详情页诊断 / 详情页文案」即可调用

### Claude Desktop / Claude.ai 网页

1. 解压 zip
2. 把 `SKILL.md` 内容复制粘贴给 Claude 作为系统提示（System Prompt）
3. 方法论文档 `references/methodology.md` 在需要时手动贴入

## 三种使用方式

### 方式一：从零生成详情页方案

```
我要做一款胶囊咖啡机的详情页：
- 品类：胶囊咖啡机（家用单杯 / 支持奶泡）
- 客单价：699 元
- 目标人群：25-35 岁一二线城市租房白领，工位上自冲咖啡省钱
- 核心参数：19 Bar 高压萃取 / 30 秒出杯 / 0.6L 水箱可拆洗
- 竞品：N 主推「咖啡馆品质」；D 主推「无线便携」；I 主推「性价比」
请生成详情页方案。
```

> ⚠️ 强制数据契约——上述 5 项缺一不开工。
> AI 拿不到这些数据就只能编，强制要求是为了避免「看起来对其实没数据支撑」的纸面分析。

### 方式二：发截屏诊断现有详情页

```
请帮我诊断这个详情页（已截屏 1-10 张按顺序贴下方）：
[贴截屏]

重点检查第一屏结构、卖点排序、因果公式、视觉规范。
```

> ⚠️ 不要给链接让 AI 自己抓——天猫/淘宝/京东都有反爬+登录墙，
> AI 抓不到内容只会编造产品信息骗你。截屏几秒钟就能搞定。

### 方式三：审查/重写现有文案

```
这是我写的详情页文案，请帮我检查：
[粘贴文案]

请按因果公式标准指出问题并给修改示例。
```

## 核心能力

✅ 强制 5 项数据契约（品类 / 客单价 / 人群 / 参数 / 竞品）
✅ 卖点挖掘四维法（对手对比 / 搜索词 / 评价痛点 / 产品属性）
✅ 12-15 屏黄金结构 + 因果文案公式
✅ 配图要求 + 视觉规范（背景色 / 字体 / 图片类型）
✅ 内置自检清单 + 数据完备度评分
✅ 经验库持久化（data/经验库.md，越用越懂你）
✅ [数据]/[推断] 来源标注，避免 AI 编造

---

## 这个 Skill 不做的事（明确边界）

❌ 不自动抓详情页链接（反爬+登录墙，技术不可能跨用户实现）
❌ 不预测转化率提升百分比（LLM 没有数据支撑）
❌ 不替你做市场调研（数据必须你提供）
❌ 不依赖任何外部 MCP、外部账号或登录态
